How it work

  • SET UP ESCROW AGREEMENT

    Setting up an escrow is easy and intuitive but, if needed, our concierge is always ready to provide assistance.

  • DEPOSIT MATERIAL

    Codekeeper integrates directly with developer workows on Github and other popular SCMplatforms. Source code deposits into the escrow are encrypted and fully automated. You are always in compliance!

  • ESCROW ACTIVE!

    Your new source code escrow is successfully activated. You can edit the details and manage other escrows as a depositor or beneciary

  • MANAGEMENT

    Easily add your licensees as beneciary of the deposited source code in escrow and keep them up to date. If you are the licensor you can add suppliers and initiate new escrows with a few clicks. Conrmations are automated. No more paperwork!
